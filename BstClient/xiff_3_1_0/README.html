XIFF: Read Me

Readme| Changelog| License

Thank you for downloading XIFF 3.1.0!

XIFF is an Open Source Flash library for instant messaging and presence clients using the XMPP (Jabber) protocol. XIFF includes an extension architecture that makes it easy to add functionality for additional protocol extensions, or even your own special-needs extensions.
